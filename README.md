# Public Library Database Management System

## Overview
This project involved designing and implementing a relational database system for a public library, including schema design, SQL implementation, and automation logic.

## Key Features
- ER diagram and relational schema design
- SQL table creation and data queries
- Borrowing and usage analytics
- Trigger-based logic for overdue alerts
- Membership status automation

## Tools Used
- PostgreSQL
- SQL
- Database design (ER modeling)

## Files Included
- ER diagram
- SQL schema and queries
- Project report
